Detailed Description

template<Hashable Key, typename Data>
class HashTable< Key, Data >

HashTable Template Class.

Hash table implementation using separate chaining as the method for hashing conflict resolution

Template Parameters
KeyHashable Key data type
DataType of data to be stored by the table
