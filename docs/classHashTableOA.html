Detailed Description

template<Hashable Key, typename Data>
class HashTableOA< Key, Data >

HashTable Template Class with Open Addressing.

Hash table implementation using Open Addressing and probing as the method for hashing conflict resolution

Template Parameters
KeyHashable Key data type
DataType of data to be stored by the table
Warning
Not thread safe
Todo:

adjust_capacity callback for better probing

Check for duplicate keys and modify the value if key exists (Probably give an optional parameter to allow for duplicates)

Bug:
Doesn't work for duplicate keys, If needed should use HashTable.
