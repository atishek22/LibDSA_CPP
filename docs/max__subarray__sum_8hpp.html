Detailed Description

Maximum Subarray Sum Algorithms.

Implementations for calculating the maximum subarray sum using std::vector. The implementation is valid for arithmetic types as defined by std::is_arithmetic<>.

Maximum Subarray Sum.

Function for finding the maximum subarray sum for the std::vector v. The function also computes the indices demarcating the required subarray.

\(O(n \log n)\)
